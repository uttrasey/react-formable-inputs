# react-formable-inputs

## Instructions

### Build

`npm run build`

### Dev

`npm start`

Then open localhost:8800 in a browser.


## Installation

**NPM**

    npm install react-formable-inputs --save

**Bower**

    bower install react-formable-inputs --save
